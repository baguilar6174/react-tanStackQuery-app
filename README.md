# React Query Examples


## Installation

Get code using

```
git clone https://github.com/baguilar6174/next-openjira-app.git
```

Step 2:

Install the necessary libraries (make sure you have node >= 16 and yarn)

```
yarn
```

Step 3:

Runs the app in the development mode

```
yarn dev
```
